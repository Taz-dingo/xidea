# Scientific Review Integration

## Positioning

Xidea 不把科学复习当作全部，而是把它当作系统里的一个子能力。

## Core Distinction

- 科学复习解决: 什么时候复习
- AI 编排解决: 现在该怎么学

## System Layers

### 学习编排层

负责决定当前动作：

- 讲解
- 追问
- 对比
- 情境模拟
- 回忆训练

### 记忆调度层

负责决定未来节奏：

- 是否需要复习
- 多久后再来
- 哪些内容优先回顾

## State Model

系统至少要区分两类状态：

### 理解状态

- understanding level
- confusion
- weak signals
- transfer / application ability

### 记忆状态

- memory strength
- last reviewed at
- next review at

## Practical Rule

- 没懂: 先教学或澄清
- 容易混淆: 先做对比训练
- 基本理解但不稳: 先做练习或迁移
- 会了但快忘: 进入科学复习节奏

## Demo Guidance

比赛阶段不需要完整实现 Anki 算法。

只要能清楚表达：

- 为什么某个学习单元现在应该先教学
- 为什么另一个学习单元现在适合复习
- 为什么系统不会把所有东西都一股脑做成卡片

就足够支撑核心论点。
